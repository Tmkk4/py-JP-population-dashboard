# py-JP-population-dashboard
SysDev A No.04

## 日本の都道府県別人口の基本統計量及び昇順・降順での都道府県別人口ランキングを表示する
### 基本統計量
- 平均値(相加平均)
- 中央値
- 最大値 / 最小値
- 分散
- 標準偏差

動作環境 : Python 3.8 

## [使用方法]
- pyファイルと同じ階層に都道府県別人口データのCSVファイルを必要とする
- 実行すると, 上記 6つの基本統計量を表示する
- 画面の指示に従い, 1:降順, 2:昇順を選択し入力することで 都道府県別人口ランキングを表示する
- 9を入力することでプログラムを終了できる
